# App Lista de Tarefas

Este projeto foi feito para colocar em prática meus aprendizados com o framework React. 


## Funcionalidas

Nele você pode:
 - Adicionar Tarefas
 - Edita-las
 - Exclui-las

## Como usar o projeto
 - Clone este repositório
 - Abra-o com sua IDE
 - No terminal da sua IDE digite: "npm i" e depois "npm start"
